6. 接入说明

以下示例与当前真实实现一致，优先使用稳定别名 `free-proxy/coding` 与 `free-proxy/auto`。
OpenAI / Python SDK
from openai import OpenAI

client = OpenAI(
    api_key="not-needed",
    base_url="http://127.0.0.1:8765/v1",
)

resp = client.chat.completions.create(
    model="free-proxy/coding",
    messages=[{"role": "user", "content": "Reply with exactly OK"}],
)
print(resp.choices[0].message.content)
OpenClaw 关键配置
provider id: free-proxy
baseUrl: http://localhost:8765/v1
models: auto, coding
默认模型: free-proxy/auto
编码场景推荐: free-proxy/coding
Opencode 验证方式
配置路径: ~/.config/opencode/opencode.json
provider id: free-proxy
推荐命令:
opencode run -m free-proxy/coding "Reply with exactly OK"

保守命令:
opencode run -m free-proxy/auto "Reply with exactly OK"
